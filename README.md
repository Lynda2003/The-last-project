##  Projet Final : Application E-Commerce Full Stack

### Description
Application e-commerce professionnelle et complète développée avec la stack **MERN**,  
intégrant toutes les fonctionnalités d'une boutique en ligne réelle.

###  Live Demo
🔗 [Voir l'application en ligne](https://mernapp-lynda-dzbqengsgzdhayfh.swedencentral-01.azurewebsites.net)

###  Fonctionnalités
-  **Authentification** — Inscription, connexion, JWT, protection des routes
-  **Catalogue produits** — Liste, recherche, filtres par catégorie et prix
-  **Panier d'achat** — Ajout, suppression, modification des quantités
-  **Système de paiement** — Intégration paiement en ligne sécurisé
-  **Gestion des commandes** — Suivi des commandes en temps réel
-  **Dashboard Admin** — Gestion produits, utilisateurs et commandes
-  **Responsive Design** — Compatible mobile, tablette et desktop

###  Technologies utilisées

| Couche | Technologie |
|--------|-------------|
| **Frontend** | React.js, HTML5, CSS3, JavaScript |
| **Backend** | Node.js, Express.js, REST API |
| **Base de données** | MongoDB, Mongoose, MongoDB Atlas |
| **Authentification** | JWT (JSON Web Tokens) |
| **Déploiement** | Microsoft Azure (App Service) |
| **Versioning** | Git, GitHub |
| **Outils** | VS Code, Postman, Canva |
